Every programmer ought to have a web presence, and this is mine. Welcome.

First and foremost, I'm currently a computer science student ('19) at the University of Illinois at Urbana-Champaign. In the past, I worked as a full-stack web developer, building applications from the bottom up using Python and AngularJS. Since June of 2016, I've been doing research in computer security with the Network Security Research Group at UIUC. Our latest work involves looking for bugs in SSL/TLS implementations by performing differential testing over the IPv4 space. In my free time, I enjoy Capture The Flag, Graphics/VFX, and going to information security conferences.

The sidebar has links to various bits of my online life. If you're a UIUCTF player looking for flags, you won't find any on this page, but the links might be a good place to start.

This site is using an adaptation of the Tufte CSS Framework. I've tried to give it a bit of flavor of my own, but frontend is not my forte, and I would sooner write my own kernel than write any more CSS than I absolutely have to.
